# 3차 프로젝트 1팀 회의록
- 날짜 2026년 03월 23일
- 팀명 : Prompt
- 참여자 : 정확,김재웅,박민성
## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- 관리자 로그인 및 플랜 연결
## 작업 회고
채팅방 메시지 조회 API에서 소유권 검증이 빠져있다는 걸 코드를 다시 살펴보다가 발견했다. 로그인만 되어 있으면 채팅방 ID만 알아도 다른 사람의 대화 내역을 볼 수 있는 보안 문제가 있었는데 , 놓치지 않고 수정할 수 있었다
## 내일목표
- 코드 정리 및 다듬기
## 팀 규칙
- ERD, 화면설계서, 명세서 는 기능이 변경 및 추가 될때마다 그때그때 바로 추가하고 팀원들 한테 고지할 것
- 다른사람이 작성하는 파일을 건드리거나 변경시 말하고 작업 시작하기 혹시 누가 작업중일 경우 잠깐 스탑 하고 다른일 먼저 처리
- 주석 달기 / 테스트 잘 실행하고 추가하기
- 야근이을 안하기 위해 최대한 노력하고 혹시나 야근이 필요할경우 야근 참석하기
- 싸우지말기
## 기술 스택
- spring boot
- JWT,JPA
- HTML,CSS,JS
- H2,SQL
