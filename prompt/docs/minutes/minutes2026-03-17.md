## 오전회의 내용
- 자기소개하기
- 팀장 뽑기 ( 정확씨 당선 )
- 팀명 정하기 (닥트리오)
- 프로젝트 명 정하기 (prompt)
- 깃헙 저장소 복제 및 피그마 ERD 초대 수락
- 작업 요구 명세서 작성
## 작업 목표
- ai 대화 api 기능 구현 및 추가
- 메인페이지 및 결제 페이지 카드 섹션 바꾸기
## 작업 회고
- Alan AI API 미구현 2개를 추가하고.
  /api/v1/question (단순 응답)과 /api/v1/question/plain-streaming을 AlanAiClient, AlanAiService, AlanAiController에 추가하고 ai-tools.html/js에 AI 질문 탭도 붙임. 테스트 코드도 기존 3개에서 5개로 늘려서 테스트
  플랜별 모델 접근 제한도 구현하였음.
  NORMAL 플랜에서 alan-4.1이나 turbo를 선택하면 업그레이드 안내 팝업이 뜨고, 확인 누르면 결제 페이지로 이동하도록 chat.js와 chat.html을 수정하였음.
