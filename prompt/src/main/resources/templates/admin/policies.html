플랜 / 정책

플랜별 제공 범위와 사용 제한 정책 관리

정책 안내

관리 대상

NORMAL, PRO, MAX 플랜별 사용 정책을 관리합니다.

정책 항목

채팅 수, 이미지 업로드 수, 파일 업로드 수, 파일 용량 제한 등을 설정합니다.

주의 사항

정책 변경 시 사용자 서비스 이용 범위에 직접 영향을 줄 수 있습니다.

NORMAL 플랜

  • 일일 채팅 제한 20회
  • 이미지 업로드 제한 5개
  • 파일 업로드 제한 3개
  • 파일 최대 용량 10MB
  • 월간 토큰 한도 50,000

PRO 플랜

  • 일일 채팅 제한 100회
  • 이미지 업로드 제한 20개
  • 파일 업로드 제한 10개
  • 파일 최대 용량 50MB
  • 월간 토큰 한도 300,000

MAX 플랜

  • 일일 채팅 제한 무제한
  • 이미지 업로드 제한 50개
  • 파일 업로드 제한 30개
  • 파일 최대 용량 100MB
  • 월간 토큰 한도 1,000,000
