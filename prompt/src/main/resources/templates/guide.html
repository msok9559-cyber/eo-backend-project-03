Prompt AI란?

  • 빠르고 안정적인 응답
  • 다양한 플랜으로 맞춤 이용

토큰이란?

토큰은 AI와 대화할 때 사용되는 단위입니다. 질문과 답변 모두 토큰을 소모하며, 매월 1일에 초기화됩니다.

AI 대화 사용법

실시간 AI 대화

알랜 AI와 자유롭게 대화할 수 있습니다. 대화의 맥락을 기억하기 때문에 이전에 나눈 내용을 바탕으로 연속적인 대화가 가능합니다.

  • 질문, 분석, 창작 등 다양한 주제로 대화 가능
  • 이전 대화 내용을 기억해 자연스러운 흐름 유지
  • 방금 말한 내용을 이어서 심화 질문 가능

페이지 요약 / 번역

웹페이지나 문서에서 복사한 텍스트를 붙여넣으면 AI가 핵심만 골라 요약해 드립니다. 긴 글도 빠르게 파악할 수 있습니다.

  • 웹페이지, 문서, 기사 등 텍스트 복사 후 붙여넣기
  • AI가 핵심 내용만 추출해 간결하게 요약
  • 원하는 언어로 번역도 함께 요청 가능

유튜브 자막 요약

유튜브 영상을 직접 보지 않아도 됩니다. 자막을 복사해 붙여넣으면 AI가 챕터별로 정리해 핵심 내용을 요약해 드립니다.

  • 유튜브 자막 텍스트 복사 후 붙여넣기
  • 챕터별로 구분해 체계적으로 요약
  • 긴 영상도 빠르게 내용 파악 가능
